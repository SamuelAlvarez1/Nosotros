📞 TROUBLESHOOTING:

❌ Fotos no cargan
  → Verifica path en config.json
  → Usa /assets/images/archivo.jpg
  → Asegúrate que exista el archivo

❌ Botón "No" no se mueve en móvil
  → Normal, movimiento es en hover (desktop)
  → En móvil funciona al tocar

❌ Animaciones lentas
  → Reduce duration en Framer Motion
  → Aumenta stiffness en spring controls

❌ Build falla
  → rm -rf node_modules
  → npm install
  → npm run build
