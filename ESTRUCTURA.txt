✅ Checklist de Personalización

□ Editar public/config.json (título, textos, mensajes)
□ Agregar fotos en public/assets/images/
□ Agregar stickers en public/assets/stickers/ (opcional)
□ Probar localmente: npm run dev
□ Verificar en móvil
□ Compilar: npm run build
□ Desplegar en GitHub Pages (o Netlify/Vercel)
□ ¡Realizar la propuesta! 💖
